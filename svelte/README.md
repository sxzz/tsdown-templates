# svelte-components-starter

A starter for creating a Svelte component library.

## Development

- Install dependencies:

```bash
npm install
```

- Run the unit tests:

```bash
npm run test
```

- Build the library:

```bash
npm run build
```

- Run the playground for development:

```bash
npm run play
```

- Type checking:

```bash
npm run typecheck
```
